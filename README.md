Thông tin nhóm:
1. Đỗ Quốc Huy
2. Hà Thế Hiển
3. Nguyễn Đỗ Khiêm



#### Framework
      Client: ReactJS
      Blockchain: Ethereum Smart Contract - Metamask - Ganache
      
#### How to run client
      1. Go to client: cd ./client
      2. Install packages: yarn install
      3. Run app: yarn start
